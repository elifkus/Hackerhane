Hackerhane
==========

Hackerspace üyeleri için uygulama. 


Planlanan özellikler:

- Üye listesi
- Üye profilleri
- Aidat ödeme bilgileri
- Single sign on
- Social sign in
- Bütçe - giderler
- Anasayfa - etkinlik takvimi, mekanın açık olup olmadığı, eğitimler, aktif projeler
- Projeler
- Eğitimler (?)


Gereklilikler
=====================
Python 3.x

Django 1.6
